# MohawkRemotePython

**To start Mohawk Server. Command line:**
```
cd C:\Program Files\Ziath\Mohawk\
```
```
MohawkServer.exe
```

```
cd C:\Program Files\Ziath\Mohawk\
```
```
MohawkServer.exe -ss
```

```
MohawkRemoteExample.py
```
